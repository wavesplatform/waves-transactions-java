# Waves-transactions for Java

Library for binary and json serialization of transactions in Waves blockchain.

## Installation

### Maven

```
<dependency>
    <groupId>com.wavesplatform</groupId>
    <artifactId>waves-transactions</artifactId>
    <version>1.0.7</version>
</dependency>
```
